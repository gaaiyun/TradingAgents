## 二、所选8项技术指标及其选择理由

| 指标 | 选择理由 |
|---|---|
| **close_10_ema** | 短期趋势捕捉，反映近期价格动量变化，适合判断入场时机 |
| **close_50_sma** | 中期趋势基准，作为动态支撑/阻力，判断中期方向 |
| **close_200_sma** | 长期趋势确认，判断整体牛市/熊市格局及均线交叉信号 |
| **macd** | 核心动量指标，通过EMA差值捕捉趋势转折，检测金叉/死叉 |
| **macdh** | MACD柱状图，可视化动量强度变化，提前发现动量背离 |
| **rsi** | 衡量超买/超卖程度，识别潜在反转信号及动量强弱 |
| **boll_ub** | 布林带上轨，标识潜在阻力区及突破/超买区域 |
| **atr** | 波动率测量，用于设定止损位和仓位管理 |

这8项指标覆盖了**趋势（短期/中期/长期）、动量、波动率、超买超卖**四个维度，彼此互补且无冗余。
